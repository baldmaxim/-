## Ключевые правила расценки

### 1. Единицы измерения — главная ловушка
- **Стены из блоков** в ВОР часто в м2, а в ГЭСН в м3 → коэффициент = толщина стены
  - Стена 300мм: 1 м2 = 0.3 м3
  - Стена 400мм: 1 м2 = 0.4 м3
- **Перегородки** обычно в м2 — коэффициент = толщина блока (0.1 или 0.15)
- **Кирпичная кладка**: 1 м2 стены в полкирпича = 51 шт кирпича

### 2. Обязательный состав кладки из блоков (м3)
- Блоки газобетонные D500 × 1.02 (2% потери)
- Клей газобетонный: 5 кг/м3
- Раствор ЦПС для первого ряда: 0.04 м3/м3
- Сетка кладочная: через каждые 3 ряда
- Арматура для перемычек и армопоясов
- Монтажная пена для заполнения швов
- U-блоки для перемычек
- Анкера/связи к монолиту: 5-7 шт/м2

### 3. Обязательные вспомогательные материалы
- Монтажная пена: 0.3 баллона/м3
- L-профиль для опирания перегородок
- Демпферная лента по периметру
- Подъём материалов краном: отдельная строка

### 4. ГЭСН код для кладки
Основной код: 08-02-001-01 (кладка стен из камней).
Единица в ГЭСН = м3 кладки.

## Типичные ошибки

1. **Перепутать м2 и м3** — кладка стен считается в м3, перегородки в м2, нужен коэффициент толщины
2. **Забыть армирование** — сетка каждые 3 ряда, арматура в армопоясах и перемычках
3. **Не учесть U-блоки** — для перемычек и армопоясов, они дороже обычных блоков
4. **Забыть связи с монолитом** — анкера, L-профиль для опирания = отдельные строки
5. **Не учесть потери** — коэффициент 1.02 на блоки (бой при транспортировке и резке)
6. **Забыть подъём материалов** — кран или ручной подъём = отдельная строка, до 500 руб/м3
7. **Считать стеклянные перегородки как кладку** — это совсем другой шаблон с другой ценой
