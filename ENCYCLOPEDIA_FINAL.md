## ⚠️ ЛОВУШКИ ЕДИНИЦ ИЗМЕРЕНИЯ — ЧИТАЙ ПЕРЕД РАСЦЕНКОЙ!

### Правило 1: В AVR базе единицы НЕНАДЁЖНЫ
Пометка "куб.м" в AVR может означать:
- **Реально руб/м³** (нужно умножить на толщину стены в метрах для получения руб/м²)
- **Ошибочная метка, реально руб/м²** (брать как есть)

**ВСЕГДА проверяй через таблицу цен выше!**

### Правило 2: Если AVR цена > RED FLAG → вероятно это м³!
```
ПРИМЕР ОШИБКИ (реальный, ЖК Муза):
AVR дал: газобетон 200мм = 11,872 руб (пометка "куб.м")
Агент применил как 11,872 руб/м² → RED FLAG > 7,000!

Правильно: 11,872 руб/м³ × 0.2 м = 2,374 руб/м² → в диапазоне ✅
Ошибка стоила +15 млн руб на 1,200 м²!
```

### Правило 3: Формулы пересчёта
- **м³ → м²:** Цена_м² = Цена_м³ × толщина_м (пример: 11,872 × 0.2 = 2,374)
- **м² → м³:** Цена_м³ = Цена_м² / толщина_м

### Правило 4: Как определить реальную единицу AVR
1. Берём AVR цену
2. Проверяем "как есть" (руб/м²) → попадает в таблицу?
3. Проверяем "как м³" (×толщину) → попадает в таблицу?
4. Тот вариант, который попадает — правильный

**Инструмент: `python tools/kladka_sanity_check.py --pricing file.json`**

## ⚠️ ЛОВУШКИ ЕДИНИЦ ИЗМЕРЕНИЯ

### Правило 1: В AVR базе единицы НЕНАДЁЖНЫ!

Пометка "куб.м" в AVR может означать:
- **Реально руб/м³** — нужно умножить на толщину для получения руб/м²
- **Ошибка метаданных** — цена уже за м², несмотря на пометку

**AVR база НЕПОСЛЕДОВАТЕЛЬНА!** Примеры из ЖК Муза:
| AVR запись | Цена | Пометка | Реально | Как определили |
|------------|------|---------|---------|----------------|
| Газобетон 200мм | 11,872 | куб.м | руб/м³ | 11,872 > RED FLAG 7000 → это м³ |
| Газобетон 100мм | 2,424 | куб.м | руб/м² | 2,424 в диапазоне 1800-3500 → это м² |
| Кирпич 120мм | 4,848 | куб.м | руб/м² | 4,848 × 0.12 = 582 → абсурд → это м² |
| ПГП 80мм | 2,424 | куб.м | руб/м² | 2,424 в диапазоне 2200-3500 → это м² |

### Правило 2: Формулы пересчёта

```
м³ → м²:  Цена_м² = Цена_м³ × толщина_м
м² → м³:  Цена_м³ = Цена_м² / толщина_м

Примеры:
  11,872 руб/м³ × 0.20м = 2,374 руб/м² (газобетон 200мм)
  11,872 руб/м³ × 0.15м = 1,781 руб/м² (газобетон 150мм)
  11,872 руб/м³ × 0.30м = 3,562 руб/м² (газобетон 300мм)
```

### Правило 3: Алгоритм определения реальной единицы

```
1. Получил цену AVR → посмотри в таблицу СВОДКА ЦЕН выше
2. Цена попадает в диапазон MIN–MAX? → это руб/м², бери как есть
3. Цена > RED FLAG? → вероятно руб/м³ → умножь на толщину
4. После умножения попадает в MIN–MAX? → подтверждено, это м³
5. Ни один вариант не подходит → бери ТИПИЧНУЮ из таблицы
```

### Правило 4: Если AVR цена > RED FLAG → ПОЧТИ НАВЕРНЯКА это м³

Газобетон 200мм по 11,872 руб/м²? → 11,872 > 7,000 (RED FLAG)
- Проверяем: 11,872 × 0.2 = 2,374 → попадает в 2,300–4,500 ✅
- РЕШЕНИЕ: AVR дала цену за м³, не за м²!

## ❌ ПРИМЕРЫ ТИПИЧНЫХ ОШИБОК (РЕАЛЬНЫЕ СЛУЧАИ ЖК МУЗА)

### Ошибка 1: Газобетон 200мм = 11,872 руб/м² (×5 завышение!)

**Что случилось:**
AVR дал 11,872 руб (sim=0.76, GREEN) с пометкой "куб.м".
Агент записал: "AVR DB хранит цены в куб.м, но ВОР использует м2. Цены применены как руб/м2".
Агент ЗНАЛ о проблеме единиц, но принял неверное решение.

**Правильно:**
11,872 руб — это цена за КУБИЧЕСКИЙ МЕТР газобетонной кладки.
Для стены 200мм: 1 м² = 0.2 м³ → 11,872 × 0.2 = 2,374 руб/м²
С корректировкой в верхнюю треть: 3,500 руб/м²

**Масштаб ошибки:** 7 позиций газобетона завышены в ~5 раз → +22 млн руб фантомных

### Ошибка 2: КСЦ 190мм = 8,000 руб/м² (×1.9 завышение!)

**Что случилось:**
AVR не нашёл совпадения (sim=0.64). Агент сделал "экспертную оценку":
- Работа: 4,000-6,000 руб/м² ← ЗАВЫШЕНО в 3 раза!
- Итого: 8,000 руб/м²

**Правильно по энциклопедии:**
- Работа КСЦ 190мм: 1,500-2,200 руб/м²
- Материал: ~910 руб/м²
- Расходники: ~490 руб/м²
- Итого: 3,200-5,000 руб/м² → верхняя треть = 4,200 руб/м²

**Масштаб ошибки:** Это КЛЮЧЕВАЯ позиция (46% бюджета, 2,311 м²) → +8.8 млн руб

### Ошибка 3: Парсинг нашёл 12 позиций вместо ~27+

**Что случилось:**
Парсер отфильтровал только раздел 1.1.x (первый этаж). Пропущены:
- Раздел 1.7.2 — ещё ~4 позиции кладки 1 этажа
- Раздел 2.3.2 — наружные стены типовых этажей (~6 позиций)
- Раздел 2.3.3 — внутренние стены типовых этажей (~6 позиций)

**Масштаб ошибки:** Расценено 4,963 м² вместо ~35,000+ м².
Пропущено 86% объёма кладки → занижение бюджета на ~80+ млн руб.

### Урок: ТРИ УРОВНЯ ЗАЩИТЫ от ошибок

```
УРОВЕНЬ 1: Ценовая матрица    → цена в диапазоне?
УРОВЕНЬ 2: Конвертация единиц → м²/м³ правильно?
УРОВЕНЬ 3: Полнота парсинга   → ВСЕ разделы ВОР просканированы?
```

- В AVR встречается редко — при расценке используй ФЕР для «кладки из блоков»

| Теплопроводность λ | 0,07–0,12 Вт/(м·К) | 0,12–0,17 Вт/(м·К) |

| Морозостойкость | F25–F50 | F25–F50 |

| Усадка | 0,4–0,5 % | — |
| Огнестойкость | 0,75–1,5 ч | — |

#### Ключевые свойства

✅ **Преимущества:**
- Самая высокая теплоизоляция среди стеновых блоков (λ от 0,07 Вт/(м·К))
- Лёгкий (400–850 кг/м³) — минимальная нагрузка на фундамент
- Хорошая звукоизоляция, хорошо обрабатывается (пилится, гвоздится)
- Высокая прочность на изгиб — «пружинит», не трескается при осадке
- Не горит (органика выгорает, цемент держит форму)

❌ **Недостатки:**
- **Высокое водопоглощение (40–85%)** — обязательна защитная штукатурка снаружи!
- Нельзя в зонах повышенной влажности без защиты
- Нестабильность качества (много мелких производителей)
- Меньше доступность по сравнению с газобетоном

#### Типовые размеры блоков
- 500×200×300 мм (стеновой), 500×150×300 мм (перегородочный)
- 600×200×400 мм (крупноформатный)

#### Области применения
- Малоэтажное жилое строительство (1–3 этажа), коттеджи
- Хозяйственные постройки, гаражи (без прямого контакта с влагой)
- Перегородки и самонесущие стены
- **НЕ применять:** при влажности воздуха >75%, ниже гидроизоляции

| Составляющая | Цена |

#### 🚩 Красные флаги
- Арболит в мокрых помещениях без защиты → разрушение (водопоглощение 40–85%!)

- Требует **тёплого раствора** — обычный ЦПС снижает теплоэффективность

## 2. НОРМЫ И ЦЕНООБРАЗОВАНИЕ

### 2.1 Структура расценки ГЭСН / ФЕР

**Принципиальная схема:**
- **ГЭСН** (Государственные элементные сметные нормы) содержит только трудовые и машинные затраты. Стоимость материалов **не включена** — учитывается отдельно ресурсным методом.
- **ФЕР** (Федеральные единичные расценки) = ГЭСН × стоимостные показатели. ФЕР уже содержит стоимость материалов.

**Ключевые сборники:**
- **ГЭСН-2017, Сборник 08** «Конструкции из кирпича и блоков»
- **ФЕР-2017, Сборник 08** (стоимостной вариант)
- **ТЕР** (территориальный — для каждого региона свой)

**Три статьи прямых затрат:**

| Статья | Обозначение | Что входит |
|--------|-------------|-----------|
| Оплата труда | ОТ | З/п каменщиков; тарифная ставка × нормочасы |
| Эксплуатация машин | ЭМ | Аренда крана, раствороузла, подъёмника (вкл. з/п машиниста) |
| Материалы | М | Кирпич, блоки, раствор, закладные (в ФЕР — по сметным ценам ФСС) |

**Доли в расценке (ФЕР, сб. 8):**
- ОТ — ~18–30% прямых затрат
- ЭМ — ~5–12%
- М — ~58–77%

**Структура сборника ГЭСН-08:**

| Номер | Вид работы |
|-------|-----------|
| 08-01 | Кладка стен из кирпича (разные толщины) |
| 08-02 | Кладка из блоков ячеистого бетона |
| 08-03 | Армирование кладки |
| 08-04 | Перемычки (укладка) |
| 08-05 | Кладка перегородок (кирпич и блоки) |
| 08-06 | Кладка в зимних условиях (прогрев, тепляк) |
| 08-07 | Подмости и леса (перестановка, устройство) |

---

### 2.2 Что входит и не входит в расценку кладки

**ЧТО ВХОДИТ в расценку ГЭСН «кладка стен»:**
- Труд каменщиков и подсобных рабочих
- Приготовление кладочного раствора (если не указано отдельно)
- Подача раствора к месту укладки
- Установка и перестановка подмостей на уровне до 1-го яруса (≤ 3 м)
- Расшивка швов
- Перемещение материалов по фронту работ (горизонтальное)

**ЧТО НЕ ВХОДИТ (оплачивается отдельно):**
- Сам кирпич / блок / плита (материал)
- Кладочный раствор или клей (материал)
- Арматурная сетка для армирования
- Доставка материалов на объект (транспортные расходы)
- Вертикальный подъём материалов (при высоте > 1 этажа — кран или подъёмник)
- Устройство подмостей/лесов при высоте > 3 м (расценки 08-07)
- Зимние мероприятия (надбавки МДС 81-33.2004)
- Накладные расходы и сметная прибыль (НР и СП к итогу прямых затрат)

---

### 2.3 Надбавки и коэффициенты к расценкам кладки

#### Высота кладки > 3 м (подмости, леса)

Расценки ГЭСН/ФЕР рассчитаны при высоте яруса **до 3 м**.

| Ситуация | Метод учёта | Коэффициент / надбавка |
|---------|-------------|------------------------|
| Кладка от 0 до 3 м | Нормативная | — (базовая расценка) |
| Кладка выше 3 м (каждый ярус) | Расценка на перестановку подмостей | ФЕР 08-07-001 ~1 чел.-ч / 1 м³ |
| Здания высотой > 25 м (башенный кран) | Коэффициент к ЭМ | К = 1,15 к маш.-часам крана |
| Кладка на лесах (фасад) | Отдельная расценка на леса | ФЕР 08-07-002..008 на 100 м² леса |

> **Практическое правило:** на каждый ярус > 3 м добавляется стоимость устройства и разборки инвентарных подмостей. При высоте здания > 3 этажей — обязательно включать леса наружные (трубчатые или рамные).

#### Зимнее удорожание

Основание: **МДС 81-33.2004**

| Тип работ | Коэффициент (базовый, р-н II Б — Центральная Россия) |
|-----------|------------------------------------------------------|
| Кладка кирпичная | **1,03–1,05** к ОТ+ЭМ (без материалов) |
| Кладка с прогревом раствора | Доп. стоимость прогрева: ~120–250 руб/м³ |
| Устройство тепляка | Отдельная расценка ~800–1 500 руб/м² ограждения |

*Коэффициент варьируется по регионам: Сибирь (р-н IV) — до 1,10; Москва (р-н II Б) — 1,03–1,04.*

#### Стеснённые условия производства работ

Основание: **МДС 81-35.2004**, п. 4.7 + Приложение 1

| Условие | Коэффициент к ОТ + ЭМ |
|---------|----------------------|
| Стеснённость в городских условиях (действующее предприятие) | **К = 1,15** |
| Вблизи действующего оборудования (опасная зона) | **К = 1,25** |
| В подземных условиях (подвалы, тоннели без освещения) | **К = 1,10–1,20** |
| Поэтапная сдача участков (прерывистое производство) | **К = 1,10** |
| Особая стеснённость (режимный объект, двор < 5 м) | **К = 1,30** |

*Коэффициенты суммируются до максимума 1,50; при совпадении нескольких — один максимальный + 0,05 за каждое доп.*

#### Прочие надбавки

| Условие | Надбавка | Основание |
|---------|---------|-----------|
| Кладка в вытяжных шахтах, вентканалах | +25–50% | ГЭСН примечания |
| Кладка в труднодоступных местах | +30–60% | По согласованию |

### 2.4 Армирование кладки

#### Когда применяется
- **Обязательно**: столбы, простенки (при нагрузке > расчётной), перемычки, сейсмические зоны (7+ баллов)
- **По проекту**: наружные стены с ветровой нагрузкой, подпорные стенки
- **Стандартно**: каждые 3–5 рядов при кладке столбов; каждые 5–6 рядов при кладке стен
- **Для газобетона**: продольные пруты в пазах через каждые 2–3 ряда

#### Нормы расхода сетчатого армирования

| Параметр | Значение |
|----------|---------|
| Периодичность укладки | Каждые **3–5 рядов** кирпичной кладки |
| Диаметр проволоки | Ø 3–5 мм (Вр-I), Ø 4–6 мм (В-I) |
| Ячейка сетки | 50×50 мм, 60×60 мм, 100×100 мм |
| Расход сетки (каждые 3 ряда) | ~0,9–1,1 м² сетки на 1 м² стены |
| Расход сетки (каждые 5 рядов) | ~0,55–0,65 м² сетки на 1 м² стены |
| Расход проволоки (d=4мм, ячейка 50×50) | ~1,5–2,5 кг/м² стены (армирование каждые 5 рядов) |

#### Типы армирования

| Тип | Описание | Норма расхода | Расценка ГЭСН |
|-----|----------|---------------|---------------|
| **Кладочная сетка Вр-I Ø3–6мм** | Горизонтальные швы, антисейсмика | 3–5 кг/м³ кладки | ГЭСН 08-01-xxx |
| **Арм. стержни А-I, А-III** | Горизонтальные пояса, армирование перемычек | 3–8 кг/м³ кладки | ГЭСН 08-03 |
| **Гибкие связи (БПС, нерж.)** | Соединение облицовочной кладки с несущей | 4–6 шт./м² фасада | ГЭСН 08-01-003/005 |
| **Армосетки заводские АIII** | Армопояс, сейсмика | 8–15 кг/м³ (по проекту) | ГЭСН 08-03 |

#### Гибкие связи — детально
- **Базальтопластиковые** (БПС): d = 4–8 мм, l = 250–500 мм
- **Из нержавеющей стали**: d = 4 мм, П-образные или якорные
- **Стандартный шаг**: 500×500 мм = 4 шт./м²
- **По СП 15.13330.2020**: min 4 шт./м² для облицовочных стен; min 6 шт./м² по периметру проёмов (≤ 200 мм от края)

**В смете:** армирование считается **отдельной позицией** (расценка 08-03 или 08-01-003/005); раздельный учёт: труд (монтаж) + материал (сетка/стержни/связи).

#### При поиске «кирпич»:
```

```

#### При поиске «раствор»:
```
раствор для стяжки (пескобетон М150–М200 — не кладочный),
раствор инъекционный (эпоксидный, для усиления),
клей плиточный (≠ кладочный раствор),
грунтовка, пропитка (ошибочно попадают)
```

#### При поиске «блок»:
```
блок-контейнер → временные здания,
блочно-модульный → модульное строительство,
блок питания → электрика,
распределительный блок → электрика,
задвижка блочная → трубопроводы,
фундаментный блок ФБС → монолит (при поиске стеновых блоков),
ФБС (фундаментный) — если нужны стеновые блоки
```

> **Безопасно уточнять**: `стеновой блок`, `кладочный блок`, `блок ячеистого бетона`.

#### При поиске «демонтаж vs монтаж»:
```
демонтаж, разборка (если ищёшь монтажные работы)
```

#### Прочие нерелевантные:
```
бутовый камень (другой раздел — бутовая кладка),

```

## 6. КРАСНЫЕ ФЛАГИ И ТИПИЧНЫЕ ОШИБКИ ПРИ РАСЦЕНКЕ

### 6.1 Ошибки в определении материала

| Ошибка | Последствие |
|--------|------------|
| Газобетон принят как пенобетон | Разная цена, разные ГОСТы, разный контроль |
| Поризованный блок (POROTHERM) принят как пустотелый кирпич | Сильно разная единичная расценка и расход |
| КСЦ принят как шлакоблок | Разные характеристики, разная цена |
| «Кирпич» без уточнения марки и типа | Невозможно корректно расценить |
| ПГП перепутана с гипсокартоном | Совершенно разные технологии и расценки |
| «Блок ячеистого бетона» без уточнения авто/неавтоклавный | Подмена газобетона пенобетоном |
| «Кирпич поризованный» (НФ) vs POROTHERM (крупноформатный) | Разные расценки и стоимость материала |

### 6.2 Ошибки в единицах измерения

| Ошибка | Как исправить |
|--------|--------------|
| Перегородки 0,5 кирпича считают в м³ | Для перегородок — м², для стен ≥ 1 кирпича — м³ || ПГП считают в м³ | Для ПГП — только м² |
| Газобетон в м² без указания толщины | Нельзя корректно расценить |
| КСЦ в шт вместо м³ | Нестандартно, пересчитать |

### 6.3 Ошибки в объёмах
- Объёмы без вычета проёмов (завышение 15–25%)
- Не учтены деформационные швы (занижение)
- Перепутаны м² фасада и м² кладки
- Армирование не выделено отдельной позицией

### 6.4 Ошибки в расценках
- Применена ФЕР без учёта региональных коэффициентов
- Не добавлены надбавки за высоту (>3м)
- Пропущено зимнее удорожание
- Двойной счёт: материал включён в ФЕР, но ещё раз в спецификации
- НР и СП не начислены на прямые затраты

### 7.3 Алгоритм анализа позиции ВОР

```
1. Прочитай наименование → определи МАТЕРИАЛ (по синонимам из раздела 3)
2. Проверь единицу измерения (раздел 2.5)
3. Определи толщину/конструктив
4. Найди расценку ГЭСН/ФЕР (сборник 08)
5. Проверь: включён ли материал в расценку?
6. Если нет — добавь позицию на материал
7. Проверь надбавки: высота, зима, стеснённость
8. Проверь армирование (отдельная позиция?)
9. Сравни итог с ценовым ориентиром (раздел 7.2)
10. Красные флаги → раздел 6
```
