# ЧЕК-ЛИСТ QA КЛАДОЧНЫХ РАБОТ
## 5 обязательных проверок перед финализацией расценки

**Версия:** 1.0 | **Дата:** 2026-02-20  
**Применение:** Независимый QA-агент (Opus) проверяет КАЖДУЮ расценку кладки перед отправкой.

---

## ✅ ПРОВЕРКА 1: ЕДИНИЦЫ ИЗМЕРЕНИЯ

**Что проверить:**
- [ ] Для каждой позиции зафиксирована единица ВОР (м² или м³)
- [ ] Для каждой цены AVR зафиксирована единица источника
- [ ] Если AVR единица ≠ ВОР единица → выполнена конвертация через толщину
- [ ] Конвертация задокументирована: `price_m2 = price_m3 × thickness_m`
- [ ] Результат конвертации проверен через price_matrix.json (попадает в range)

**Типичная ошибка:** AVR даёт 11,872 руб за газобетон — это за м³! За м² при 200мм = 11,872 × 0.2 = 2,374.

**Красный флаг:** Цена газобетона/ПГП/КСЦ > 7,000 руб/м² → почти наверняка цена в м³ не конвертирована.

---

## ✅ ПРОВЕРКА 2: ДИАПАЗОН ЦЕН (SANITY CHECK)

**Что проверить:**
- [ ] Каждая цена сверена с price_matrix.json
- [ ] Ни одна цена не превышает `red_flag_above` для своего материала+толщины
- [ ] Ни одна цена не ниже `red_flag_below`
- [ ] Средняя цена по ВСЕМ позициям кладки в диапазоне 3,100–5,000 руб/м²
- [ ] Бюджетообразующие позиции (>20% итого) проверены ТРИЖДЫ

**Формула средней:** `Σ(price_i × volume_i) / Σ(volume_i)`

**Контрольные диапазоны (руб/м², комплексно):**
| Материал | Толщина | Допустимо | Red Flag |
|----------|---------|-----------|----------|
| Газобетон D500 | 100мм | 1,800–3,500 | >5,000 |
| Газобетон D500 | 200мм | 2,300–4,500 | >7,000 |
| Кирпич полнот. | 120мм | 2,200–4,000 | >6,000 |
| КСЦ | 190мм | 3,200–5,000 | >8,000 |
| ПГП/ПГПВ | 80мм | 2,200–4,000 | >6,000 |

---

## ✅ ПРОВЕРКА 3: ПОЛНОТА ПОЗИЦИЙ

**Что проверить:**
- [ ] Просканированы ВСЕ разделы ВОР (не только 1.1.x)
- [ ] Проверены разделы: 1.1.x, 1.7.x, 2.3.x, 2.6.x, и любые другие
- [ ] Для каждого раздела: проверены ВСЕ подразделы на ключевые слова кладки
- [ ] Типовые этажи (раздел 2.x) учтены с правильными множителями
- [ ] Объём в м³ (типовые этажи) корректно пересчитан в м²
- [ ] Общий объём кладки разумен для проекта (м² на секцию × кол-во этажей)

**Контрольная формула объёма:**
- ЖК на 200 квартир: ~15,000–30,000 м² кладки всего
- На 1 секцию (70-100 квартир): ~6,000–12,000 м²
- На 1 типовой этаж: ~300–600 м²

**Красный флаг:** Общий объём < 5,000 м² для ЖК с 7+ секциями = точно пропущены позиции.

---

## ✅ ПРОВЕРКА 4: НАДБАВКИ И КОЭФФИЦИЕНТЫ

**Что проверить:**
- [ ] Наружные стены — надбавка ×1.05–1.10 применена
- [ ] Шахты (стеснённость) — надбавка ×1.10–1.20 применена
- [ ] EI45/EI60 (противопожарность) — надбавка ×1.10–1.20 применена
- [ ] Высота >3м — надбавка ×1.05–1.15 если применимо
- [ ] Межквартирные стены — надбавка ×1.05–1.10 для звукоизоляции
- [ ] Надбавки НЕ накапливаются бесконтрольно (макс. суммарный коэфф. ≤ 1.35)
- [ ] Каждая надбавка задокументирована с обоснованием

**Типичная ошибка:** Применить 3 надбавки по ×1.15 = итого ×1.52. Цена выходит за red_flag.

---

## ✅ ПРОВЕРКА 5: ИТОГОВАЯ АРИФМЕТИКА И ЛОГИКА

**Что проверить:**
- [ ] Сумма = Цена × Объём — пересчитана для каждой строки
- [ ] Итого = Σ всех строк — сошлось
- [ ] Доля кладки в общем бюджете ≤ 15% (для типового ЖК)
- [ ] Нет дублей (одна и та же стена расценена дважды из разных разделов ВОР)
- [ ] Для типовых этажей: проверить, не расценён ли уже тот же объём в разделе 1.x

**Контрольное соотношение:**
- Кладка / Общий бюджет ЖК ≈ 8–15%
- Если кладка > 20% общего бюджета → проверить (вероятно завышены цены)
- Если кладка < 5% общего бюджета → проверить (вероятно пропущены позиции)
