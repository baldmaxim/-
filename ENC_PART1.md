## МАТЕРИАЛ 1: КСЦ / СКЦ — КЕРАМЗИТОБЕТОННЫЙ СТЕНОВОЙ КАМЕНЬ

> **Ключевая путаница в поле:** КСЦ и СКЦ — это **один и тот же материал**. Разница только в порядке слов в аббревиатуре. В AVR могут быть обе версии — искать нужно обе.

### 1.5 Как распознать КСЦ в ВОР заказчика

**Ключевые слова (искать В НАИМЕНОВАНИИ):**

```
КСЦ, СКЦ, керамзитоблок, керамзитобетон, 
"блок 390×190", "блок 390×190×190", "камень стеновой",
"блок на керамзите", "керамзитобетонный камень",
"лёгкобетонный блок", КЛБ, КСПР
```

**Тип единицы измерения:**
- Стены → м³ (при толщине 190 мм и более)
- Перегородки 90 мм → м² или м³ (уточнять)
- Иногда в ВОР указано в м² — это стена 190 мм, тогда 1 м² = 0,19 м³

### 1.6 Красные флаги при расценке КСЦ

| Ситуация | Риск | Действие |
|---------|------|---------|
| В ВОР написано «блок 390×190×190» без уточнения материала | Может быть шлакоблок или бетонный блок, а не КСЦ | Уточнить у заказчика состав и ГОСТ |
| В AVR нашлась расценка на «шлакоблок» вместо КСЦ | Разные характеристики и цены | Использовать отдельные расценки |
| Марка М50 для несущих стен многоэтажки | Недостаточная прочность | Требовать конструктивный расчёт |
| КСЦ применён вместо газобетона по теплотехнике | Λ керамзитоблока выше → стена холоднее | Проверить теплотехнический расчёт стены |
| В смете «кладка блоков» без разграничения по типу | Нельзя расценить точно | Требовать спецификацию: марка, тип (КСЦ-1/2, КСПР) |

### 2.5 Как распознать в ВОР

**Ключевые слова:**
```
кирпич, глиняный, красный, рядовой, полнотелый, пустотелый,
```

**Единицы в ВОР:**
- Перегородки 0,5 кирпича → **м²** (иногда м³, надо уточнять)

- Облицовка 0,5 кирпича → **м²**

### 2.6 Красные флаги при расценке кирпича

| Ситуация | Риск | Действие |
|---------|------|---------|
| Силикатный кирпич в фундаменте/цоколе | Технически недопустимо (разрушается при постоянной влажности) | Указать ошибку заказчика |
| Силикатный кирпич для дымоходов/печей | Разрушается при T > 500°C | То же |
| Марка М300+ для рядовых стен жилого дома | Завышение стоимости материала | Уточнить у конструкторов |
| «Кирпич полнотелый» без указания марки | Невозможно корректно расценить | Запросить уточнение |
| Двойной счёт: кирпич и в расценке, и отдельно | Завышение | Проверить состав ФЕР |
| Надбавка за высоту не учтена при кладке > 3 м | Занижение сметы | Добавить надбавку |
| Кладка пустотелого кирпича во влажных зонах | Недопустимо по СП | Указать |

## МАТЕРИАЛ 3: ПГП — ПАЗОГРЕБНЕВЫЕ ПЛИТЫ (в том числе влагостойкие)

| **Плита пазогребневая гипсовая** | ГОСТ Р 51829-2001 | Официальное |

| **ПГПВ** | Производители | Плита Гипсовая Пазогребневая Влагостойкая |
| **Блок пазогребневый** | ВОР, прайсы | Синоним плиты |
| **Гипсоблок** | Бытовое, стройрынок | Неточно (гипсоблок бывает и без паза), но часто = ПГП |
| **Перегородочная плита гипсовая** | Старые сметы | Описательное |
| **Гипсовая перегородочная плита** | ВОР заказчика | |
| **KNAUF ПГП** | Торговая марка | Наиболее распространённый производитель |
| **VOLMA-Плит** | Торговая марка | Российский аналог Knauf |

| **ПГП 100** | Прайсы | По толщине 100 мм (усиленная) |
| **Влагостойкая ПГП (зелёная)** | По цвету | Визуальное отличие от стандартной |
| **Гипсовый блок для перегородок** | ВОР | |

> ⚠️ **Путаница 1:** ПГП ≠ гипсокартон (ГКЛ/ГКЛВ). Это принципиально разные технологии. ПГП — кладка, ГКЛ — обшивка каркаса. Расценки и нормы совершенно разные.  
> ⚠️ **Путаница 2:** Стандартная ПГП ≠ влагостойкая ПГПВ. В санузлах обязательно ПГПВ (зелёная). Если в ВОР не указан тип — уточняй.

### 3.2 Технические характеристики

| Параметр | Стандартная ПГП | Влагостойкая ПГПВ |
|----------|-----------------|-------------------|
| **ГОСТ** | ГОСТ Р 51829-2001 | ГОСТ Р 51829-2001 |
| **Стандарт производителя** | ТУ Knauf, VOLMA | ТУ Knauf, VOLMA |
| **Размер Knauf (осн. серия)** | 667 × 500 × 80 мм | 667 × 500 × 80 мм |
| **Размер «усиленная»** | 667 × 500 × 100 мм | 667 × 500 × 100 мм |
| **Масса плиты 80 мм** | ~24 кг | ~24–26 кг |
| **Плотность** | 1 050–1 100 кг/м³ | 1 050–1 100 кг/м³ |
| **Предел огнестойкости** | EI 30–45 мин | EI 30–45 мин |
| **Звукоизоляция (80 мм, одинарная)** | ~37–41 дБ | ~37–41 дБ |
| **Звукоизоляция (80 мм, двойная)** | ~47–50 дБ | ~47–50 дБ |
| **Цвет** | Серый / белый | **Зелёный** (гидрофобные добавки) |
| **Водостойкость** | Низкая — разрушается при намокании | Высокая — пригодна для влажных помещений |
| **Предел прочности на сжатие** | ≥ 5,0 МПа | ≥ 5,0 МПа |

**Расход материала:**
- 1 м² перегородки = **3 плиты** Knauf 667×500 (покрытие: 0,667×0,500 = 0,33 м²/шт)
- Клей монтажный: ~1,5–2 кг на 1 плиту → ~4,5–6 кг/м²
- Дюбели: 3–4 шт/м² (крепление к полу и потолку)

### 3.3 Области применения

| Тип перегородки | Нужная плита | Примечание |

### 3.5 Как распознать ПГП в ВОР

**Ключевые слова:**
```
ПГП, ПГПВ, пазогребневая, пазогребневые, гипсоблок,

ВОЛМА, "667×500", "плита 80мм", "плита 100мм"
```

**Единицы в ВОР:**
- Всегда **м²** (реже м³, но это нестандартно)
- Объём в ВОР = площадь перегородки в свету

**Отличие от смежных позиций в ВОР:**
| Если написано... | Это... |
|-----------------|--------|
| «Перегородки из ПГП» | Кладка ПГП — наш раздел |
| «Перегородки из ГКЛ / гипсокартона» | Другой раздел (ГКЛ по каркасу) |
| «Перегородки кирпичные» | Кирпич — другая расценка |
| «Перегородки из газобетона» | Газобетонный блок — другая расценка |

### 3.6 Красные флаги при расценке ПГП

| Ситуация | Риск | Действие |
|---------|------|---------|
| Стандартная ПГП (не ПГПВ) в санузлах / кухнях | Разрушение при намокании → переделка | Требовать замену на ПГПВ |
| ПГП в несущих стенах | Технически недопустимо | Указать ошибку |
| Монтаж без демпферной (уплотнительной) ленты | По инструкции обязательна — без неё трещины | Уточнить состав работ |
| Монтаж ПГП на ЦПС (цементный раствор) | По инструкции — только специальный клей | Ошибка технологии |
| В ВОР «кладка ПГП» без учёта клея | Занижение материалов | Добавить позицию «клей монтажный» |
| ПГП учтена в м³ (как кирпич) | Нестандартно, пересчитай объём | Привести к м² |
| Двойная перегородка ПГП — учтена как одинарная | Занижение объёма вдвое | Уточнить конструктив |

## МАТЕРИАЛ 4: ГАЗОБЕТОН АВТОКЛАВНЫЙ (D500, D600)

### 4.1 Синонимы и аббревиатуры

| Термин | Источник | Пояснение |

> ⚠️ **Важная путаница: автоклавный vs неавтоклавный**  
> Автоклавный газобетон (наш случай) — заводской, стабильное качество, точная геометрия (±1 мм).  
> Неавтоклавный (пенобетон) — часто кустарный, хуже геометрия, выше усадка.  
> Если в ВОР написано просто «блок ячеистого бетона» — уточни: автоклавный или нет? Цена и расценка разные.

### 4.5 Как распознать газобетон в ВОР

**Ключевые слова:**
```
газобетон, газосиликат, газоблок, ячеистый бетон, автоклавный,
D400, D500, D600, D700, B2,5, B3,5,
Ytong, Bonolit, H+H, ИНСИ, Hebel,
"блок 600×300", "блок 600×200", "блок ячеистый"
```

**Отличие «автоклавный» vs «неавтоклавный» в ВОР:**
- Если указано «автоклавный» / «D500» / конкретная торговая марка → наш раздел
- Если написано «пеноблок» / «неавтоклавный» / «монолитный ячеистый» → другой материал (пенобетон)
- Если просто «блок ячеистого бетона» без уточнения → **обязательно уточни у заказчика**

**Единицы в ВОР:**
- Стены → **м³** (объём кладки с учётом проёмов — вычитаются)
- Перегородки → **м³** или **м²** (с указанием толщины)
- Иногда встречается в м² — уточни толщину для пересчёта в м³

### 4.6 Красные флаги при расценке газобетона

| Ситуация | Риск | Действие |
|---------|------|---------|
| Газобетон D400 в несущих стенах многоэтажки | Недостаточная прочность | Требовать конструктивный расчёт |
| Кладка на ЦПС вместо тонкошовного клея | Мостики холода через швы — ухудшение теплозащиты | Уточнить в спецификации; если ЦПС — другая единичная расценка |
| Газобетон в фундаменте/цоколе | Разрушается при намокании | Недопустимо; указать ошибку |
| В ВОР не разграничено D500 и D600 | Разная цена материала (~15%) | Запросить спецификацию по маркам |
| Расчёт кладки без вычета проёмов | Завышение объёма (иногда до 20–25%) | Проверить ведомость проёмов |
| Клей кладочный не выделен отдельной позицией | Занижение материалов | Добавить позицию |
| Газобетон = пенобетон в смете | Подмена (пенобетон дешевле, хуже) | Уточнить производителя, ГОСТ |
