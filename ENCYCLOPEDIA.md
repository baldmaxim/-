## 1. МАТЕРИАЛЫ КЛАДКИ — ПОЛНЫЙ СПРАВОЧНИК

### ПРИНЦИПЫ РАБОТЫ С РАЗДЕЛОМ

Перед анализом ВОР установи:
- **Тип материала** (из наименования позиции)
- **Единица измерения** (м2 — кладка в полкирпича; м3 — кладка от 1 кирпича и выше)
- **Включён ли материал в расценку** (ФЕР/ТЕР — обычно нет, нужно смотреть примечание)
- **Надбавки**: высота, зима, стеснённость

#### Как распознать в ВОР
Ключевые слова: `кирпич`, `глиняный`, `красный`, `рядовой`, `полнотелый`, `НФ`, `керамический`, `обожжённый`.  
Единицы: м² (перегородки, облицовка), м³ (стены ≥ 1 кирпича).

#### 🚩 Красные флаги
- Марка М300+ для рядовых стен → завышение
- В ВОР указана «кладка» — уточни, включён ли кирпич в расценку
- Кладка фундамента из силикатного кирпича → нарушение (силикат нельзя во влажную зону)
- «Кирпич полнотелый + армирование» через каждый ряд → явное завышение (норма: каждые 3–5 рядов)

## 2. НОРМЫ И ЦЕНООБРАЗОВАНИЕ

### 2.1 Система ГЭСН/ФЕР для кладочных работ

**Ключевые сборники:**
- **ГЭСН-2017, Сборник 08** «Конструкции из кирпича и блоков»
- **ФЕР-2017, Сборник 08** (стоимостной вариант ГЭСН-08)
- **ТЕР** (территориальный — для каждого региона свой)

**Структура расценки в ГЭСН-08:**

| Номер | Вид работы |
|-------|-----------|
| 08-01 | Кладка стен из кирпича (разные толщины) |
| 08-02 | Кладка перегородок |
| 08-03 | Кладка столбов и простенков |
| 08-04 | Кладка из блоков ячеистого бетона |
| 08-05 | Кладка из пазогребневых плит |
| 08-06 | Армирование кладки |
| 08-07 | Надбавки (высота, сложность) |

### 2.2 Состав расценки «Кладка стен»

**ЧТО ВХОДИТ в расценку ГЭСН «кладка стен»:**
- Труд каменщиков и подсобных рабочих
- Приготовление кладочного раствора (если не указано отдельно)
- Подача раствора к месту укладки
- Установка и перестановка подмостей на уровне до 1-го яруса (≤ 1,2 м)
- Расшивка швов
- Перемещение материалов по фронту работ (горизонтальное)

**ЧТО НЕ ВХОДИТ (оплачивается отдельно):**
- Сам кирпич / блок / плита (материал)
- Кладочный раствор или клей (материал)
- Арматурная сетка для армирования
- Доставка материалов на объект (транспортные расходы)
- Вертикальный подъём материалов (при высоте > 1 этажа — нужны башенный кран или подъёмник)
- Устройство подмостей/лесов при высоте > 1-го яруса (расценки 08-07, 09-04)
- Зимние мероприятия (надбавки)

### 2.3 Надбавки к расценкам

| Условие | Надбавка | Основание |
|---------|---------|-----------|
| Высота 4–6 м (2-й ярус подмостей) | +15–20% к труду | ГЭСН-08, примечание |
| Высота 6–9 м (3-й ярус) | +25–35% к труду | ГЭСН-08 |
| Высота > 9 м (леса) | Расценки сборника 09 | Отдельная позиция |
| Зимнее производство работ | +6–15% к прямым затратам | ГСН 81-05-02 |
| Стеснённые условия (коэф. до 1,35) | +10–35% | МДС 81-35.2004 |
| Кладка в вытяжных шахтах, вентканалах | +25–50% | ГЭСН примечания |
| Кладка в труднодоступных местах | +30–60% | По согласованию |

**Условия применения надбавки «стеснённые условия»:**
- Расстояние до стен/конструкций < 1 м
- Работа в действующих цехах
- Ограниченная высота помещения
- Наличие действующих коммуникаций вблизи

### 2.4 Армирование кладки

**Нормативные требования (СП 15.13330.2020):**
- Армирование кладки устраивается по расчёту
- При сейсмике 7 баллов и выше — обязательно
- Типовое армирование — через каждые **3–5 рядов** кирпича по высоте (или по проекту)
- Для газобетона — армирование продольными прутами в пазах через каждые 2–3 ряда

**Типы армирования:**

| Тип | Применение | Расход |
|-----|-----------|--------|
| Сетка кладочная Вр-1, ячейка 50×50 мм | Кирпич, шаг 3–5 рядов | 0,3–0,7 кг/м² кладки |
| Сетка кладочная, ячейка 100×100 мм | Блоки крупного формата | 0,2–0,4 кг/м² |
| Арматура А-III диам. 6–8 мм (в пазах) | Газобетон, поризованная керамика | 0,8–1,5 кг/пм кладки |
| Базальтопластиковая арматура | Современные решения, коррозиестойко | 0,5–1,0 кг/пм |

**Расход сетки Вр-1 ø3мм ячейка 50×50:**
- На 1 м² кладки (1 кирпич): ~0,5 кг при шаге армирования через 3 ряда
- На 1 м³ кладки: ~2,8 кг

**Как учитывать в смете:**
- Армирование считается **отдельной позицией** (расценка 08-06 или аналог)
- Раздельный учёт: труд (монтаж сетки) + материал (сетка)

## 4. СТОП-СЛОВА ДЛЯ AVR-ПОИСКА

Позиции AVR, содержащие эти слова — **нерелевантные** для кладочного раздела, их нужно отфильтровать.

### 4.1 Дорожное строительство (путаница «бетон», «блок», «кладка»)

| Стоп-слово | Почему нерелевантно |
|-----------|---------------------|
| `автомобильная дорога` | Дорожные работы |
| `аэродромный` | Аэродромное покрытие |
| `монолитный дорожный` | Монолитный бетон для дорог |
| `дорожная плита` | Сборные ж/б плиты для дорог |
| `асфальтобетон` | Дорожное покрытие |
| `бордюрный камень` | Дорожный элемент |

### 4.4 Путаница терминов «блок» в разных разделах

Слово «блок» встречается в:
- Кладочные блоки (нужно) — газобетон, КСЦ, ПГП
- Блок-контейнер → временные здания
- Блочно-модульный → модульное строительство
- Блок питания → электрика
- Распределительный блок → электрика
- Задвижка блочная → трубопроводы
- Фундаментный блок ФБС → монолит

**Используй уточняющие слова**: `стеновой блок`, `кладочный блок`, `блок ячеистого бетона`.

## 6. КРАСНЫЕ ФЛАГИ И ТИПИЧНЫЕ ОШИБКИ ПРИ РАСЦЕНКЕ

### 6.1 Ошибки в определении материала

| Ошибка | Последствие |
|--------|------------|
| Газобетон принят как пенобетон | Разная цена, разные ГОСТы, разный контроль |
| Поризованный блок (POROTHERM) принят как пустотелый кирпич | Сильно разная единичная расценка и расход |
| КСЦ принят как шлакоблок | Разные характеристики, разная цена |
| «Кирпич» без уточнения марки и типа | Невозможно корректно расценить |
| ПГП перепутана с гипсокартоном | Совершенно разные технологии и расценки |

### 6.2 Ошибки в единицах измерения

| Ошибка | Как исправить |
|--------|--------------|
| Перегородки 0,5 кирпича считают в м³ | Для перегородок — м², для стен ≥ 1 кирпича — м³ |
| Объём кладки посчитан без вычета проёмов | Уточнить у ПИР площадь проёмов |
| Газобетон 300 мм считают как газобетон 200 мм | Проверить толщину по рабочим чертежам |

### 6.3 Ошибки в составе расценки

| Ошибка | Последствие |
|--------|------------|
| Материал включён в расценку и вынесен отдельно (двойной счёт) | Завышение сметы на стоимость материала |
| Надбавка «высота» не учтена при кладке выше 3 м | Занижение сметы |
| Армирование не выделено отдельной позицией | Занижение сметы |
| Зимние мероприятия не учтены | Занижение при зимнем строительстве |
| Кладочный клей для газобетона не учтён отдельно | Занижение материалов |

### 6.4 Ошибки AVR-поиска

| Ошибка | Как избежать |
|--------|-------------|
| AVR нашёл «кирпич дорожный» вместо стенового | Используй стоп-слова: `дорожный`, `тротуарный` |
| AVR нашёл монолитные работы с бетоном | Стоп-слово: `монолитный`, `заливка`, `опалубка` |
| AVR нашёл «блок» — электротехнический | Добавь уточнение: `стеновой`, `кладочный` |
| Найдены расценки на демонтаж вместо монтажа | Стоп-слово: `демонтаж`, `разборка` |
| Расценки из другого региона | Всегда указывай регион или пересчитывай через индексы |

## 7. ШПАРГАЛКА: РАСПОЗНАТЬ МАТЕРИАЛ В ВОР

### 7.1 По ключевым словам в наименовании

```
Встретил в ВОР → определяй материал так:

"кирпич" + "полнотелый"/"рядовой"/"красный"    → Кирпич керамический полнотелый
"кирпич" + "пустотелый"/"дырчатый"              → Кирпич керамический пустотелый
"кирпич" + "силикатный"/"белый"                 → Кирпич силикатный
"кирпич" + "клинкерный"/"клинкер"               → Кирпич клинкерный
"блок" + "газобетон"/"газосиликат"/"ячеистый"   → Газобетон автоклавный
"блок" + "пенобетон"/"пеноблок"                 → Пенобетон
"блок" + "керамзит"/"КСЦ"/"СКЦ"                → Керамзитобетонный камень
"блок" + "шлак"/"шлакобетон"/"КСШ"             → Шлакоблок
"блок" + "керамический"/"поризованный"          → Поризованный блок (POROTHERM)
"плита" + "пазогребневая"/"ПГП"/"гипс"          → ПГП
"блок" + "арболит"/"древобетон"                 → Арболит
"блок" + "теплоблок"/"многослойный"             → Теплоблок
```

## ПРИЛОЖЕНИЕ Б. ГЛОССАРИЙ

| Термин | Определение |
|--------|------------|
| **ВОР** | Ведомость объёмов работ — перечень работ с объёмами от заказчика |
| **AVR** | Автоматизированная выборка расценок — поиск расценок в базе |
| **ГЭСН** | Государственные элементные сметные нормы |
| **ФЕР** | Федеральные единичные расценки |
| **ТЕР** | Территориальные единичные расценки |
| **НФ** | Нормальный формат (кирпич 250×120×65 мм) |
| **Верста** | Крайний (лицевой) ряд кирпичной кладки |
| **Забутка** | Внутреннее заполнение кладки между верстовыми рядами |
| **Штраба** | Ступенчатый уступ в кладке при перерыве работ |
| **Тычок** | Кирпич уложен короткой торцевой гранью наружу (250×65) |
| **Ложок** | Кирпич уложен длинной боковой гранью наружу (250×120) |
| **Расшивка** | Обработка видимых швов кладки специальным инструментом |
| **Перевязка** | Система чередования рядов для монолитности кладки |
| **ЦПС** | Цементно-песчаный строительный раствор |
| **Теплопроводность λ** | Вт/(м·°C) — чем меньше, тем теплее стена |
| **Марка прочности М** | Нагрузка на сжатие в кгс/см², которую выдерживает материал |
| **Морозостойкость F** | Количество циклов замораживания/оттаивания без разрушения |
