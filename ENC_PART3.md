## 1. Состав расценки кладки по ГЭСН / ФЕР

### 1.1 Принципиальная схема

Расценка ГЭСН (Государственные элементные сметные нормы) содержит **только трудовые и машинные затраты**. Стоимость материалов в ГЭСН НЕ включена — они учитываются отдельно ресурсным методом.

ФЕР (Федеральные единичные расценки) = ГЭСН × стоимостные показатели. ФЕР уже содержит стоимость материалов в составе расценки (т.е. «полная» цена за ед. измерения).

### 1.2 Три статьи прямых затрат в расценке

| Статья | Обозначение | Что входит | Примечание |
|--------|-------------|-----------|------------|
| Оплата труда | ОТ | Заработная плата рабочих-строителей (каменщики) | Тарифная ставка × нормочасы |
| Эксплуатация машин | ЭМ | Аренда крана, раствороузла, подъёмника | Включая з/п машиниста |
| Материалы | М | Кирпич, блоки, раствор, закладные детали | В ФЕР — по сметным ценам ФСС |

### 1.3 Структура расценки ФЕР-2001, сборник 8 «Конструкции из кирпича и блоков»

**Единица измерения:** 1 м³ кладки (большинство позиций).

В базе ФСНБ-2001 (редакция 2017/2020) расценки отдела 8 имеют следующую структуру прямых затрат:

- **ОТ** — составляет ~ 18–30% от прямых затрат (зависит от вида кладки)
- **ЭМ** — ~ 5–12% (кран башенный или стреловой + раствороузел)
- **М** — ~ 58–77% (кирпич + раствор — основные статьи)

> **Важно:** В расценках ГЭСН 08-xx «материалы» прописаны по *количеству* (расходу), но не по цене. Цены на материалы сметчик привязывает из текущих сборников (ФСС, ТССЦ, конъюнктурный анализ).

### 1.4 Что расценка кладки НЕ включает (учитывается отдельно)

- Устройство и разборка лесов / подмостей (отдельная расценка ФЕР 08-07-xxx)
- Транспортировка материалов на объект (ФЕР-2001, сб. 1 или транспортная составляющая в цене материала)
- Зимнее удорожание (коэффициент МДС 81-35.2004, гл. 9)
- Накладные расходы и сметная прибыль (НР и СП применяются к итогу прямых затрат)

## 2. Надбавки и коэффициенты к расценкам кладки

### 2.1 Высота кладки > 3 м (подмости, леса)

Расценки ГЭСН/ФЕР на кладку рассчитаны при высоте яруса **до 3 м** (рабочий стоит на полу или подмостях).

| Ситуация | Метод учёта | Коэффициент / надбавка |
|----------|-------------|----------------------|
| Кладка на высоте от 0 до 3 м | Нормативная | — (базовая расценка) |
| Кладка выше 3 м (каждый следующий ярус) | Добавляется расценка «перестановка подмостей» | ФЕР 08-07-001 ~ 1 чел.-ч / 1 м³ на перестановку |
| Здания высотой > 25 м (башенный кран) | Коэффициент к ЭМ | К = 1,15 к стоимости маш.-часов крана |
| Кладка на лесах (фасад) | Отдельная расценка на леса | ФЕР 08-07-002..008, расценка на 100 м² леса |

> **Практическое правило:** На каждый ярус кладки > 3 м добавляется стоимость устройства и разборки инвентарных подмостей. При высоте здания > 3 этажей обязательно включать в смету леса наружные (трубчатые или рамные).

### 2.2 Зимнее удорожание

Основание: **МДС 81-33.2004** (Методика определения стоимости строительной продукции с учётом зимних удорожаний).

| Тип работ | Коэффициент зимнего удорожания (базовый, климатический р-н II Б — Центральная Россия) |
|-----------|--------------------------------------------------------------------------------------|
| Кладка кирпичная | **1,03–1,05** к прямым затратам ОТ+ЭМ (без материалов) |
| Кладка с прогревом раствора | Дополнительно стоимость прогрева: ~120–250 руб./м³ (или отдельная расценка) |
| Устройство тепляка | Отдельная расценка, ~ 800–1500 руб./м² ограждения |

> Коэффициент варьируется по регионам: для Сибири (р-н IV) — до 1,10; для Москвы (р-н II Б) — 1,03–1,04. Применяется к разделу сметы пропорционально зимнему периоду работ или к итогу сметы единым коэффициентом по таблицам МДС 81-33.2004.

### 2.3 Стеснённые условия производства работ

Основание: **МДС 81-35.2004**, п. 4.7; **Приложение 1 к МДС**.

| Условие | Коэффициент к ОТ + ЭМ |
|---------|----------------------|
| Стеснённость в городских условиях (действующее предприятие, ограниченный фронт) | **К = 1,15** |
| Вблизи действующего оборудования (в пределах опасной зоны) | **К = 1,25** |
| В подземных условиях (подвалы, тоннели без естественного освещения) | **К = 1,10–1,20** |
| Поэтапная сдача участков (прерывистое производство) | **К = 1,10** |
| Особая стеснённость (режимный объект, узкий двор < 5 м) | **К = 1,30** |

> Коэффициенты суммируются только до 1,50 (ограничение МДС); при совпадении нескольких условий — применяется один максимальный + 0,05 за каждое дополнительное.

## 3. Армирование кладки

### 3.1 Когда применяется

- Обязательно: столбы, простенки (при нагрузке > расчётной), перемычки, сейсмические зоны
- По проекту: наружные стены с ветровой нагрузкой, подпорные стенки
- Стандартно: каждые 3–5 рядов при кладке столбов; каждые 5–6 рядов при кладке стен

### 3.2 Сетчатое армирование (кладочная сетка)

| Параметр | Значение |
|----------|---------|
| Периодичность укладки | Каждые **3–5 рядов** кирпичной кладки (через 1 или 2 ряда по высоте шва) |
| Для кирпича нормального (65 мм + 10 мм шов) | 3 ряда = 225 мм; 5 рядов = 375 мм |
| Для газобетонных блоков (200 мм) | Каждые 2–3 ряда = 400–600 мм |
| Диаметр проволоки | Ø 3–5 мм (Вр-I — холоднотянутая), Ø 4–6 мм (В-I) |
| Ячейка сетки | 50×50 мм, 60×60 мм, 100×100 мм |
| Расход сетки | **0,9–1,1 м²** сетки на **1 м²** стены (армирование каждые 3 ряда) |
| Расход сетки (каждые 5 рядов) | **0,55–0,65 м²** сетки на **1 м²** стены |
| Нормативный расход проволоки | ~0,8–1,2 кг/м² стены (при d=4мм, ячейке 50×50) |

> Расчёт: если сетка 4 Вр-I, ячейка 50×50 мм → масса 1 м² ≈ 1,0 кг. При укладке каждые 3 ряда кирпича (высота ряда 77 мм × 3 = 231 мм ≈ 4,3 сетки на 1 м высоты стены) расход ~4,3 × 1,0 кг ≈ 4,3 кг/м² → слишком много; практически расход 1,5–2,5 кг/м² стены (армирование каждые 5 рядов).

### 3.3 Типы армирования и нормы расхода

| Тип | Описание | Норма расхода | Расценка ГЭСН |
|-----|----------|---------------|---------------|
| **Кладочная сетка** Вр-I Ø3–6мм | Горизонтальные швы, антисейсмика, столбы | 3–5 кг/м³ кладки | ГЭСН 08-01-xxx (в составе кладки) |
| **Арматурные стержни** А-I, А-III | Горизонтальные пояса, армирование перемычек, лигатуры | 3–8 кг/м³ кладки | ГЭСН 08-03 (армирование отдельной строкой) |
| **Гибкие связи** (базальтопластик, нержавейка) | Соединение наружной облицовочной кладки с несущей стеной | 4–6 шт./м² фасада (или по проекту) | ГЭСН 08-01-003 / 08-01-005 (в составе облицовки) |
| **Арматурные сетки заводские** АIII | Армопояс, сейсмика (зоны 7+ баллов) | По проекту, 8–15 кг/м³ | ГЭСН 08-03 |

### 3.4 Гибкие связи — дополнительно

- **Базальтопластиковые** (БПС): d = 4–8 мм, l = 250–500 мм, шаг 500×500 мм
- **Из нержавеющей стали**: d = 4 мм, П-образные или якорные
- **Стандартный шаг**: 500 мм по горизонтали × 500 мм по вертикали = 4 шт./м²
- **Норма для СП 15.13330.2020**: min 4 шт./м² для облицовочных стен, min 6 шт./м² по периметру проёмов (на расстоянии ≤ 200 мм)
- **В ГЭСН**: учитываются в расценке на «кладку с облицовочным слоем» или отдельной позицией

## 5. Стоп-слова для поиска в базе АВР (нерелевантные результаты)

При поиске материалов кладки в базе AVR (или аналогичных ценовых базах) следующие слова/фразы в наименовании позиции **означают нерелевантный результат** — их нужно исключать.

### 5.1 Дорожное строительство и аэродромы

```
аэродром, аэродромный, ВПП, взлётно-посадочная полоса,
дорожный, дорожная одежда, асфальтобетон, асфальт,
щебень фракционированный для дорог, основание дороги,
бортовой камень, поребрик (дорожный), брусчатка (дорожная),
плита дорожная ПАГ, ПДН, дорожная плита
```

### 5.2 Монолитные конструкции

```
монолит, монолитный, опалубка, опалубочный,
бетон товарный (марка П-3 и выше — подвижный для монолита),
арматурный каркас пространственный, вязальная проволока (для монолита),
гидроизоляция монолитного пола, праймер (кровельный), мастика кровельная,
плёнка полиэтиленовая (подстилающая для монолита)
```

### 5.3 Кровельные и изоляционные работы

```
кровельный, кровля, рубероид, ондулин, металлочерепица,
профнастил кровельный, конёк кровельный, водосток,
мягкая кровля, битум кровельный, наплавляемая гидроизоляция
```

### 5.4 Отделочные работы (штукатурка, гипс)

```
гипсокартон, ГКЛ, ГКЛВ, потолочная система, подвес, профиль CD/UD,
шпатлёвка финишная, краска водно-дисперсионная, обои,
штукатурка фасадная декоративная (короед, барашек), керамическая плитка для пола
```

### 5.5 Инженерные сети и коммуникации

```
труба стальная (водопроводная), фитинг, фланец, задвижка,
кабель, провод, лоток кабельный, гофра,
воздуховод, вентиляционный короб, фасонина вентиляционная
```

### 5.6 Деревянные и металлические конструкции

```
балка деревянная, стропила, лаги, доска обрезная (стропильная),
металлоконструкция, колонна стальная, балка стальная двутавр, швеллер,
профильная труба (конструктивная), уголок стальной (несущий)
```

### 5.7 Специфические стоп-слова при поиске «кирпич»

```
клинкер тротуарный (≠ кирпич для кладки),
кирпич огнеупорный ША/ШБ (только для печей, не для стен),
кирпич силикатный (при поиске керамического и наоборот),
плитка под кирпич (декоративная),
кирпич бетонный декоративный (≠ конструктивный)
```

### 5.8 Специфические стоп-слова при поиске «раствор»

```
раствор для стяжки (пескобетон М150–М200 — не кладочный),
раствор инъекционный (эпоксидный, для усиления),
клей плиточный (≠ кладочный раствор),
клей для газобетона (специальный тонкошовный — отдельная категория),
грунтовка, пропитка (ошибочно попадают в «раствор»)
```
