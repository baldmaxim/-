# ПРАВИЛА ПАРСИНГА ВОР: КЛАДОЧНЫЕ РАБОТЫ
## Как найти ВСЕ позиции кладки в ведомости объёмов работ

**Версия:** 1.0 | **Дата:** 2026-02-20  
**Уроки из:** ЖК Муза — пропущено 60%+ позиций из-за неполного парсинга

---

## 1. КАКИЕ РАЗДЕЛЫ ВОР СОДЕРЖАТ КЛАДКУ

### 1.1 Типичная структура ВОР жилого комплекса (Мангазея и аналоги)

```
Раздел 1 — Первый этаж / Подземная часть
├── 1.1   — Стены и перегородки (КЛАДКА! ✅)
├── 1.2   — Конструкции перекрытий
├── ...
├── 1.7   — Дополнительные работы
│   ├── 1.7.2 — Стены и перегородки кладочные (КЛАДКА! ✅)
│   └── ...
└── 1.x   — прочие

Раздел 2 — Типовой этаж (× N этажей!)
├── 2.3   — Стены и перегородки
│   ├── 2.3.2 — Наружные кладочные (КЛАДКА! ✅)
│   ├── 2.3.3 — Внутренние кладочные (КЛАДКА! ✅)
│   └── ...
├── 2.6   — Прочие работы
│   └── 2.6.5 — Возможна кладка (проверить! ⚠️)
└── ...

Раздел 3+ — Кровельный этаж / техэтаж
├── 3.x   — Могут быть парапеты, шахты (проверить! ⚠️)
└── ...
```

### 1.2 КРИТИЧЕСКИЙ УРОК: Раздел 2.x = 80-90% объёмов кладки!
В ЖК Муза мы расценили только раздел 1.1 (4,963 м²) и пропустили раздел 2.3 (~30,000+ м²).
Типовые этажи содержат ОСНОВНОЙ объём кладки жилого дома.

**Правило:** ВСЕГДА начинать парсинг с раздела 2.x (типовые этажи), потом 1.x.

## 2. КЛЮЧЕВЫЕ СЛОВА ДЛЯ ПОИСКА

### 2.1 ВКЛЮЧИТЬ — ищем эти слова в описании позиции

**Материалы (прямое совпадение):**
```
кирпич          (любой контекст, кроме стоп-слов)
газобетон        газосиликат     газосиликатный
пенобетон        пенобетонный
ПГП              пазогребнев     пазогребневый
КСЦ              КСР             керамзитобетон     керамзитобетонный
шлакоблок        шлакобетон
Ytong            Bonolit         Aerostone       HEBEL
POROTHERM        Поротерм        поризованный камень
арболит          арболитовый
теплоблок
```

**Работы (косвенное):**
```
кладка           кладочн         кладочный
блок + стен      блок + перегород
камень + стен    камень + кладоч
полнотелый + кирпич
полнотелый + блок
полнотелый + камень
```

**Толщины (подтверждение — если рядом с материалом):**
```
80мм    100мм   120мм   150мм   190мм
200мм   250мм   300мм   380мм   510мм
в полкирпича        в 1/2 кирпича       (= 120мм)
в кирпич            в один кирпич       (= 250мм)
в полтора кирпича   в 1.5 кирпича       (= 380мм)
```

### 2.2 ИСКЛЮЧИТЬ — стоп-слова (ложные совпадения)

```
клинкерная плитка под кирпич    → фасадная облицовка, НЕ кладка
керамогранит под кирпич         → облицовка
утеплитель + кирпич             → утеплитель, НЕ кладка
керамзит + стяжка               → устройство полов
керамзит + засыпка              → утепление
фахверк                         → металлоконструкции (НО: связано с кладкой!)
кирпичная крошка                → подсыпка
кирпичный бой                   → подсыпка
перемычка (отдельная строка)    → ЖБ изделие (НО: расход на кладку!)
```

### 2.3 СВЯЗАННЫЕ ПОЗИЦИИ (не кладка, но учитывать)

```
перемычки (ЖБ, металл)         → расход на дверные/оконные проёмы
фахверки стальные               → усиление кладки
армирование кладки              → может быть отдельной позицией
анкеровка кладки                → крепление к монолиту
сетка кладочная                 → армирование швов
```

## 3. ПРАВИЛА ОБРАБОТКИ ОБЪЁМОВ

### 3.1 Единицы измерения в ВОР

| Единица в ВОР | Что это | Пересчёт |
|----------------|---------|----------|
| м² | Площадь стены | Использовать как есть |
| м³ | Объём кладки | Пересчитать: м² = м³ / толщина_м |
| шт | Штуки блоков/кирпичей | Редко. Пересчитать через расход на м² |
| 100 м² | Сотни квадратных метров | Умножить на 100 |
| 1000 шт | Тысячи штук | Пересчитать: м² = 1000 / расход_шт_на_м² |

### 3.2 Множественные колонки объёмов

В ВОР Мангазеи каждая секция/этаж = отдельная колонка:
```
Колонка 10: Общий объём (может быть 0!)
Колонки 11-20: Объёмы по секциям/этажам
```

**ПРАВИЛО:** Если колонка 10 = 0, СУММИРОВАТЬ колонки 11-20.
Пример: 2.3.3.4 газобетон 200мм:
- Кол.10 = 0
- Кол.11-19 = 588.9 + 626.8 + 617.8 + 452.0 + 849.6 + 827.2 + 492.4 + 7.3 + 111.0
- Итого = 4,573 м³ → при 200мм = 22,865 м²

### 3.3 Типовые этажи — умножение

Если раздел 2.x = "типовой этаж" и в ВОР указано количество этажей:
- Объём 1 типового этажа × количество типовых этажей
- Проверить: не суммированы ли уже объёмы в колонках?

**ВНИМАНИЕ:** В некоторых ВОР объёмы по колонкам уже = все этажи × все секции.
В других — объём на 1 этаж, и нужно умножать. Определять по контексту!

## 4. ГРУППИРОВКА И ДЕДУПЛИКАЦИЯ

### 4.1 Уникальные виды кладки
Группировать позиции по: `материал + толщина + тип_стены + особенности`

Пример группировки:
```
ГРУППА: Газобетон D500 200мм внутренние стены
├── 1.1.2.1 — МОП 1 этаж: 530 м²
├── 2.3.3.4 — Типовые этажи: 22,865 м²
└── ИТОГО: 23,395 м² → ОДНА цена для всей группы
```

### 4.2 Проверка на дубли
- Одна и та же стена может быть в разделе 1.x (1 этаж) и 2.x (типовые)
- Проверить: 1 этаж = типовой? Если да — НЕ суммировать (уже учтён)
- Проверить: подвал = отдельно от 1 этажа? Если да — суммировать

### 4.3 Результат: единый реестр
```
| № | Материал | Толщина | Тип | Объём м² | Разделы ВОР | Цена/м² | Сумма |
```

## 5. КОНТРОЛЬНЫЕ ПРОВЕРКИ ПАРСИНГА

### 5.1 Плотность кладки на проект
| Тип проекта | Кладка м² | Ориентир |
|-------------|-----------|----------|
| ЖК 1 секция, 15 эт | 5,000–10,000 | |
| ЖК 3 секции, 20 эт | 20,000–45,000 | |
| ЖК 7 секций, 20 эт | 50,000–100,000 | |
| Коттедж 200 м² | 400–800 | |

### 5.2 Баланс материалов
Для типового ЖК:
- Газобетон: 40–60% объёма кладки
- КСЦ/кирпич (межквартирные): 20–35%
- ПГП (санузлы): 5–15%
- Кирпич (общий): 10–20%

### 5.3 Красные флаги парсинга
- [ ] Общий объём < 5,000 м² для ЖК > 5 секций → ПРОПУЩЕНЫ ТИПОВЫЕ ЭТАЖИ
- [ ] Нет ни одной позиции из раздела 2.x → ПРОПУЩЕН ТИПОВОЙ ЭТАЖ
- [ ] Все единицы м², но некоторые числа > 1000 → может быть м³ (проверить)
- [ ] Только 1 вид газобетона при ЖК > 3 секций → вероятно пропущены толщины

## 6. ЧЕК-ЛИСТ ПАРСИНГА (ОБЯЗАТЕЛЬНО)

Перед переходом к расценке:
- [ ] Просканированы разделы 1.x, 2.x, 3.x
- [ ] Найдены позиции во ВСЕХ подразделах кладки
- [ ] Объёмы суммированы по колонкам (не только колонка 10)
- [ ] Единицы определены (м² или м³)
- [ ] м³ пересчитаны в м² через толщину
- [ ] Нет дублей (один объём не посчитан дважды)
- [ ] Общий объём разумен для масштаба проекта
- [ ] Все виды материалов найдены (газобетон, кирпич, КСЦ, ПГП)
