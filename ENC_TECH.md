# ТЕХНОЛОГИИ КЛАДОЧНЫХ РАБОТ — СПРАВОЧНИК ЭКСПЕРТА

> Нормативная база: СП 15.13330.2020, СП 70.13330.2012, ГОСТ 8478-81

---

## 1. ВИДЫ ПЕРЕВЯЗКИ КИРПИЧНОЙ КЛАДКИ

### 1.1 Однорядная (цепная)

Тычковые и ложковые ряды чередуются через каждый ряд.

| Параметр | Значение |
|---|---|
| Смещение поперечных швов | ¼ кирпича |
| Смещение продольных швов | ½ кирпича |
| Трудоёмкость | Высокая |
| Прочность | Максимальная |

**Применение:** несущие стены, столбы, простенки, конструкции в сложных условиях (вибрация, сейсмика).

### 1.2 Многорядная (ложковая)

5–6 ложковых рядов перекрываются одним тычковым.

**Применение:** стены жилых зданий, заполнение каркаса. Производительность выше на 20–25%.

**Ограничение:** нельзя для столбов сечением < 2×2 кирпича.

### 1.3 Трёхрядная

Три ложковых → один тычковый. Применяется для перегородок 120 мм.

### 1.4 Декоративная (фасадная)

| Вид | Описание |
|---|---|
| Американская (Running bond) | Только ложковые, смещение ½ |
| Фламандская (Flemish bond) | Тычок–ложок в одном ряду |
| Английская (English bond) | Чередование тычковых и ложковых рядов |
| Голландская | Тычок + два ложка |
| Крестовая | Вертикальные швы совпадают через ряд |
| Дикая (рваная) | Смещение ¼–¾ произвольно |
| Шахматная | Тычковые ряды смещены на ½ |

## 2. ТИПЫ ЗАПОЛНЕНИЯ ШВОВ

### 2.1 Нормативные размеры швов (СП 15.13330.2020)

| Тип шва | Норма | Допуск |
|---|---|---|
| Горизонтальный | **12 мм** | ±2 мм (10–14 мм) |
| Вертикальный | **10 мм** | ±2 мм (8–12 мм) |

### 2.2 Впустошовку

Раствор не доходит до лицевой поверхности на 10–15 мм.

**Когда:** под штукатурку, облицовку плиткой, декоративную расшивку.  
**Запрещено:** на открытых фасадах без отделки.

### 2.3 Вподрезку

Раствор заподлицо с лицевой поверхностью. Применяется при кладке «под чистый кирпич».

### 2.4 Расшивка

| Тип | Описание | Применение |
|---|---|---|
| Выпуклая | Профиль выступает из плоскости | Декоративная кладка |
| Вогнутая | Профиль углублён | Лицевая кладка фасадов |
| Прямоугольная | Срезана заподлицо | Стандартная лицевая кладка |
| Треугольная | Срезана под углом 45° | Декоративная |

**Когда расшивают:** по свежему раствору (схватился, но ещё не затвердел) — через 30–60 мин после укладки ряда.

## 3. КЛАДКА В ЗИМНИХ УСЛОВИЯХ (ниже +5°C)

### 3.1 Метод замораживания

Кладка ведётся на подогретом растворе, замерзает в теле кладки, оттаивает весной.

**Условия применения:**
- Температура не ниже −30°C
- Марка раствора на 1–2 ступени выше проектной (при t < −10°C — на 2 ступени)
- Запрещено: для столбов и простенков ≤ 0,5 м²; высота кладки за смену ≤ 1,2 м

**Риски при оттаивании:**
- Снижение прочности до 50% от проектной
- Необходимо временное крепление и наблюдение
- Нагрузки до набора прочности — не более 25% расчётной

### 3.2 Противоморозные добавки

| Добавка | Дозировка | Температура применения |
|---|---|---|
| Нитрит натрия (НН) | 5–8% от массы цемента | До −15°C |
| Поташ (К₂СО₃) | 5–15% от воды затворения | До −30°C |
| Нитрит-нитрат кальция (ННК) | 3–5% | До −20°C |
| Комплексные добавки (HiCon, Freeze) | По инструкции | До −25°C |

> ⚠️ Поташ нельзя применять с алюминием, цинком и оцинкованными закладными.

### 3.3 Электропрогрев кладки

Электроды (стержни d=6 мм) укладываются в горизонтальные швы через 2–3 ряда, подключаются к трансформатору 36–127 В.  
Потребление: 150–200 Вт/м² прогреваемой кладки.

### 3.4 Тепляки

Временные ограждения (плёнка/брезент + калориферы) поддерживают t ≥ +5°C.  
Экономически оправданы при объёме кладки > 200–300 м³.

## 4. КЛАДКА НА КЛЕЙ VS НА РАСТВОР

| Материал | Вид вяжущего | Толщина шва | Почему именно так |
|---|---|---|---|
| **Кирпич** | ЦПС М75–М100 | 10–12 мм | Стандарт по СП 15 |
| **Газобетон (автоклавный)** | Тонкошовный клей | **1–3 мм** | Точная геометрия блока ±1 мм |
| **Пенобетон** | ЦПС или клей | 8–15 мм | Менее точная геометрия |
| **ПГП** | Монтажный гипсовый клей (Перлфикс) | 5–10 мм | По инструкции производителя |
| **КСЦ (керамзитоблок)** | ЦПС М75 | 10–12 мм | Стандарт |
| **Поризованный блок (POROTHERM)** | Тёплый клей λ ≤ 0,24 Вт/(м·К) | 10–12 мм | Иначе мостики холода через шов |
| **Арболит** | Теплоизоляционный ЦПС | 10–15 мм | Поглощает воду — нужно увлажнение |

### Почему газобетон НЕЛЬЗЯ класть на обычный ЦПС

Шов 10–12 мм из ЦПС (λ = 0,5 Вт/м·К) на фоне газобетона (λ = 0,11 Вт/м·К) создаёт «мостики холода». Расчёт: при стене 300 мм и шаге блоков 250 мм горизонтальные швы занимают ~5% площади, но создают теплопотери как если бы 40% стены было без утепления. Тонкошовный клей (2 мм) снижает мостики холода в 5–6 раз.

## 5. АРМИРОВАНИЕ КЛАДКИ

### 5.1 Когда применяется

- **Обязательно:** столбы (при нагрузке > расчётной), простенки, сейсмика 7+ баллов
- **По проекту:** наружные стены при высокой ветровой нагрузке
- **Стандартно:** каждые 3–5 рядов при кладке столбов; каждые 5–6 рядов при кладке стен

### 5.2 Кладочная сетка (ГОСТ 8478-81)

| Параметр | Значение |
|---|---|
| Диаметр проволоки | Ø 3–5 мм (Вр-I) |
| Ячейка | 50×50, 60×60, 100×100 мм |
| Шаг укладки | Каждые 3–5 рядов кирпича |
| Расход (каждые 3 ряда) | 0,9–1,1 м² сетки / м² стены |
| Расход (каждые 5 рядов) | 0,55–0,65 м² сетки / м² стены |
| Расход в кг (d=4, 50×50) | 1,5–2,5 кг/м² стены |

### 5.3 Гибкие связи

**Назначение:** соединение облицовочной кладки с несущей стеной.

| Параметр | Значение |
|---|---|
| Материал | Нержавеющая сталь или базальтопластик (БПС) |
| Диаметр | d = 4–8 мм |
| Длина | 250–500 мм |
| Стандартный шаг | 500×500 мм = **4 шт./м²** |
| По СП 15 (мин.) | 4 шт./м²; 6 шт./м² по периметру проёмов (≤ 200 мм от края) |

### 5.4 Армопояс

Горизонтальная ж/б балка поверх кладки перед укладкой перекрытий.  
Назначение: распределение точечных нагрузок от перекрытия.  
Минимальное сечение: 150×150 мм. Арматура: 4 стержня Ø10–12 мм А-III.

## 6. ДЕФОРМАЦИОННЫЕ ШВЫ

| Тип шва | Шаг | Причина | Заполнение |
|---|---|---|---|
| Температурно-усадочный | 20–30 м (кирпич), 40–60 м (газобетон) | Температурные деформации здания | Пенополиуретан или минвата |
| Осадочный | По расчёту | Разные нагрузки / разные грунты | Полиуретановый герметик |
| Антисейсмический | По расчёту | Сейсмика 8–9 баллов | Пенополиуретан |

**Конструкция шва:**
- Ширина: 10–20 мм
- Шов должен пронизывать всю кладку насквозь (снаружи до внутренней поверхности)
- Заполняется сжимаемым материалом, сверху — упругий герметик
- Температурный шов НЕ прерывает фундамент (только надземную часть)

## 8. АНКЕРОВКА КЛАДКИ К Ж/Б КАРКАСУ

### 8.1 Зачем анкеровать

Кладка и каркас имеют разные коэффициенты температурного расширения и разную жёсткость. Без связей — кладка отслаивается и падает.

### 8.2 Гибкие анкеры

| Параметр | Значение |
|---|---|
| Тип | Г-образные, П-образные из нержавеющей стали или БПС |
| Вертикальный шаг | **600 мм** (каждые 4 ряда кирпича = 4×65+4×12 = 308 мм ≈ через 2–3 ряда газобетона) |
| Горизонтальный шаг | 600–800 мм |
| Способ крепления к каркасу | Забивной анкер в бетон / закладные пластины |

### 8.3 Зазор между кладкой и колонной

- **Зазор:** 20 мм
- **Заполнение:** минватой или пенополистиролом (разрыв мостика холода)
- **Запрещено:** жёсткая связь кладки с колонной — при осадке кладка трескается

### 8.4 Требования СП 15.13330.2020

- Связи через каждые 4 ряда кирпичной кладки по высоте
- Анкеры в горизонтальных швах на всю длину, выступание в кладку ≥ 120 мм

## 9. КОНТРОЛЬ КАЧЕСТВА КЛАДКИ (СП 70.13330.2012)

### 9.1 Допуски на отклонения

| Показатель | Допуск |
|---|---|
| Отклонение от вертикали (на 1 этаж) | ≤ **10 мм** |
| Отклонение от вертикали (на всё здание) | ≤ **30 мм** |
| Неровность поверхности (на 2 м рейку) | ≤ **10 мм** |
| Отклонение от горизонтали (на 10 м) | ≤ **15 мм** |
| Смещение осей кладки | ≤ **10 мм** |
| Размеры проёмов | ±**15 мм** |
| Толщина горизонтальных швов (средняя на 10 рядов) | 10–13 мм |

### 9.2 Приёмка скрытых работ

Обязательна до закрытия следующим конструктивом:
- Армирование кладки (сетки, стержни) — акт на скрытые работы
- Закладные детали и связи
- Деформационные швы
- Гидроизоляция в теле кладки

### 9.3 Входной контроль материалов

- Кирпич/блоки: паспорт качества + выборочная проверка на прочность (ГОСТ 24544)
- Раствор: протокол испытания контрольных кубиков 7×7×7 см
- Арматура: сертификат + проверка диаметра

## 10. ТИПИЧНЫЕ НАРУШЕНИЯ ПРИ ПРОИЗВОДСТВЕ КЛАДОЧНЫХ РАБОТ

| Нарушение | Последствие | Как обнаружить |
|---|---|---|
| Раствор нанесён заранее — подсох | Плохое сцепление, снижение прочности до 30–40% | Простукивание, акт скрытых работ |
| Кирпич не увлажнён в жару (+25°C) | Впитывает воду из раствора, шов пористый | Звук при простукивании |
| Неполное заполнение вертикальных швов | Снижение прочности на сжатие, продувание стен | Визуальный контроль |
| Газобетон на ЦПС вместо клея | Мостики холода — теплопотери +40% | Тепловизионное обследование |
| Кладка в мороз без мероприятий | Расслоение кладки при оттаивании | Наблюдение в период оттепели |
| Жёсткая связь кладки с каркасом | Трещины при осадке здания | Визуальный контроль после 1 зимы |
| Силикатный кирпич в цоколе/фундаменте | Разрушение при намокании | Экспертиза материала |
| Пустотелый кирпич в фундаменте | Разрушение при замерзании воды в пустотах | Проверка проекта |
| Облицовочный кирпич без гибких связей | Обрушение облицовки | Проверка наличия связей при кладке |
| Тонкошовный клей для газобетона нанесён зубчатым шпателем > 3 мм | Перерасход + ухудшение геометрии | Контроль толщины шва |
